# How to install

1.  Navigate to **Blender Preferences** -> **Add-ons** tab and select the dropdown.

    ![Preferences Add-ons](assets/images/1.%20Preferences_Add-ons_Install_Add_ons.png)

2.  From the dropdown select the option **“Install from Disk…”**.

    ![Install from Disk](assets/images/1.Preferences_add-ons_Intall_from_disk_Context_Menu.png)

3.  Navigate to where you have downloaded the `.zip` file for the add-on.

4.  Select it and then click **“Install from Disk”**.

    ![Select Zip](assets/images/1.Choose_Horizontal_Tabs_Add-Ons_Install_From_Disk.png)

## Once installed

All your add-ons from the sidebar will appear in Blender’s header.

If you have a lot of add-ons you can scroll on the header with the mouse wheel to go through them.
